  数据覆盖范围（与 db.Seed() 保持一致，并做了扩充）
                                                                                                                                            
  ┌────────────┬─────────────────────────────────────────────────────────┐
  │     表     │                          数据                           │                                                                  
  ├────────────┼─────────────────────────────────────────────────────────┤                                                                
  │ tenants    │ 1 个 demo 租户（plan: pro）                             │
  ├────────────┼─────────────────────────────────────────────────────────┤
  │ users      │ admin / seller / buyer 各 1 个                          │
  ├────────────┼─────────────────────────────────────────────────────────┤
  │ products   │ 6 个商品，含 1 个低库存（stock=3）用于触发 FindLowStock │
  ├────────────┼─────────────────────────────────────────────────────────┤
  │ orders     │ buyer 的 1 笔已完成订单（2 个 line item）               │
  ├────────────┼─────────────────────────────────────────────────────────┤
  │ audit_logs │ admin 角色分配的审计记录                                │
  └────────────┴─────────────────────────────────────────────────────────┘

  幂等性设计：所有 INSERT 都用 WHERE NOT EXISTS 保护，整体包在一个事务里，重复执行不会报错也不会产生重复数据。

  使用方式：
  # 迁移 + 种子一步到位
  make migrate-up && make seed

  # 或直接指定数据库
  make seed DB_URL=postgres://root:abc23456@host:5432/showcase?sslmode=disable
